# docker常用指令

## Linux

### 啟動docker
```
systemctl start docker
```

### 停止docker
```
systemctl stop docker
```

### 重啟docker
```
systemctl restart docker
```

### 查看docker狀態
```
systemctl status docker
```

### 開機啟動
```
systemctl enable docker
```

### 查看docker概要
```
docker info
```

### 查看docker總體說明文件
```
docker --help
```

### 查看docker命令說明文件
```
docker <command> --help
```
